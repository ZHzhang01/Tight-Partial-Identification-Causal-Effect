# Tight Partial Identification and Robust Decisions with Marginal Distributions of Unmeasured Confounders

This repository contains the code used to reproduce the computational figures and tables in the paper:

**Tight Partial Identification and Robust Decisions with Marginal Distributions of Unmeasured Confounders**

The experiments use only the observed joint distribution \(P(X,Y)\) and the marginal confounder distribution \(P(U)\), as described in Section 6 and Appendix O of the paper.

---

## 1. Requirements

The code was tested with Python 3.10+.

Install dependencies by

```bash
python -m venv .venv
source .venv/bin/activate
python -m pip install --upgrade pip
pip install numpy pandas matplotlib scipy tqdm jupyter
```

Optional:

```bash
pip install cvxpy
```

`cvxpy` is only needed if one wants to run the original Jiang et al. baseline scripts directly.

From the repository root, run

```bash

```

## 5. Reproducing Figure 5

Figure 5 is generated in `Simulation.ipynb`.

The relevant setting is

```python
SEED = 20260512
FAST_MODE = True
N_SAMPLES_FAST = 20_000
N_SAMPLES_FULL = 1_000_000
N_SAMPLES = N_SAMPLES_FAST if FAST_MODE else N_SAMPLES_FULL
DU = 5
ALPHA_U = 0.1
```

The current paper figure uses

```python
FAST_MODE = True
N_SAMPLES = 20_000
```

The generated files are

```text
outputs/figures/o1_entropy_distribution.pdf
outputs/figures/figure5_nonvanilla_counts.pdf
outputs/figures/information_loss_entropy_baseline.pdf
outputs/tables/synthetic_entropy_summary.csv
outputs/tables/figure5_summary.tex
```

For a larger Monte Carlo run, set

```python
FAST_MODE = False
```

before executing the notebook.

## 10. Expected output checks

After running `Simulation.ipynb`, the following files should exist:

```text
outputs/tables/synthetic_entropy_summary.csv
outputs/tables/table6_event_bounds.csv
outputs/tables/table7_prop12_ate_bounds.csv
outputs/figures/figure5_nonvanilla_counts.pdf
outputs/figures/figure6_simplex_reproduction.pdf
```

For the archived FAST_MODE run of Figure 5, `synthetic_entropy_summary.csv` should contain 10 entropy bins:

```text
[0.0,0.1], [0.1,0.2], ..., [0.9,1.0]
```

For Table 6, the output should contain 18 event-level rows.

For Table 7, the output should contain 9 contrast rows.
